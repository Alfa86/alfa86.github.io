  • Uvod u HTML

    HTML je jezik za obilježavanje koji koristimo za izradu web stranica.
    Njime definiramo strukturu i sadržaj web stranica.

    Povijest HTML-a

    HTML je nastao početkom 1990-ih godina.

    Uređivanje teksta

    Tekst je najčešći sadržaj web stranica.
    Da bi dobio odgovarajuću ulogu na stranici i bio prikazan na određeni način uređujemo ga HTML-om.

    HTML atributi

    Unutar početne oznaka elemenata, nakon naziva elementa, po potrebi pišemo i atribute.
    Atributi internetskim preglednicima daju dodatne informacije o elementu. O samom atributu ovisi o kojim informacijama je riječ.
    Atribut se sastoji od imena i vrijednosti atributa.
